# 즐순이 - 즐겨찾기 매니저 사용자 매뉴얼

나만의 즐겨찾기를 카테고리별로 정리하고, 링크마다 메모를 남길 수 있는 웹 앱입니다.

> **PDF로 저장:** 앱 메인 화면 가운데 패널 맨 아래 **「매뉴얼 PDF」** 링크를 클릭하면 매뉴얼 페이지가 열립니다. 해당 페이지에서 **「PDF로 저장」** 버튼을 누른 뒤, 인쇄 대화상자에서 대상을 **PDF로 저장**으로 선택하면 매뉴얼을 PDF 파일로 다운로드할 수 있습니다.

---

## 1. 시작하기

### 1.1 로그인

1. 사이트에 접속합니다.
2. **「Google로 로그인」** 버튼을 클릭합니다.
3. Google 계정으로 로그인하면 메인 화면(대시보드)으로 이동합니다.

### 1.2 로그아웃

- 화면 오른쪽 상단의 **「로그아웃」** 버튼을 클릭하면 로그아웃됩니다.

---

## 2. 화면 구성

메인 화면은 **헤더**와 **세 개의 패널**로 이루어져 있습니다.

| 영역 | 설명 |
|------|------|
| **헤더** | 로고, 구글 검색·앱 검색, 이메일·로그아웃 |
| **왼쪽 패널** | 카테고리 목록 (폴더 구조) |
| **가운데 패널** | 링크 목록 (선택한 카테고리의 링크) |
| **오른쪽 패널** | 메모 목록 (선택한 링크의 메모) |

---

## 3. 헤더

### 3.1 로고

- **즐순이** 로고를 클릭하면 **처음 화면**으로 돌아갑니다.  
  (카테고리·링크 선택이 해제되고, 가운데에 명언이 다시 표시됩니다.)

### 3.2 구글 검색

- **입력창**에 검색어를 입력한 뒤 **🔍 버튼**을 누르거나 **Enter**를 치면,  
  Google 검색 결과가 **새 탭**에서 열립니다.

### 3.3 카테고리·링크·메모 검색

- **입력창**에 단어를 입력한 뒤 **🔍 버튼**을 누르거나 **Enter**를 치면,  
  **카테고리 이름**, **링크(제목·URL·설명)**, **메모 내용** 중 해당 단어가 포함된 링크만 가운데 패널에 표시됩니다.
- **「검색 해제」** 버튼을 누르면 검색이 취소되고, 다시 카테고리별 링크 보기로 돌아갑니다.
- 검색 결과 링크에는 **어느 카테고리**에 속하는지 보라색 뱃지로 표시됩니다.

---

## 4. 카테고리 (왼쪽 패널)

### 4.1 카테고리 추가

1. **「새 카테고리 이름」** 입력창에 이름을 입력합니다.
2. **「추가」** 버튼을 클릭하거나 **Enter**를 누릅니다.  
   → 최상위(루트) 카테고리가 만들어집니다.

### 4.2 하위 카테고리 추가

1. 하위 카테고리를 넣고 싶은 카테고리 **오른쪽 ➕ 버튼**을 클릭합니다.
2. 나타난 입력창에 **하위 카테고리 이름**을 입력합니다.
3. **「추가」** 또는 **Enter**를 누릅니다.

### 4.3 카테고리 이름 수정

1. 해당 카테고리에 마우스를 올리면 **✏️ 버튼**이 보입니다.
2. ✏️를 클릭한 뒤, 입력창에 새 이름을 입력하고 **「저장」**을 누릅니다.

### 4.4 카테고리 삭제

1. 해당 카테고리에 마우스를 올리면 **🗑️ 버튼**이 보입니다.
2. 🗑️를 클릭한 뒤, 확인 메시지에서 **확인**을 누르면  
   그 카테고리와 **그 안의 모든 하위 카테고리·링크·메모**가 삭제됩니다.

### 4.5 카테고리 순서 변경

1. 카테고리에 마우스를 올리면 **⬆️(위로)** / **⬇️(아래로)** 버튼이 보입니다.
2. **⬆️**를 누르면 같은 레벨에서 한 칸 위로, **⬇️**를 누르면 한 칸 아래로 이동합니다.  
   (맨 위/맨 아래에서는 해당 버튼이 보이지 않습니다.)

### 4.6 카테고리 선택

- **카테고리 이름**을 클릭하면 해당 카테고리가 선택되고,  
  가운데 패널에 그 카테고리에 속한 **링크 목록**이 표시됩니다.
- **▶ / ▼** 를 클릭하면 하위 카테고리 목록이 접히거나 펼쳐집니다.

---

## 5. 링크 (가운데 패널)

### 5.1 링크 추가

1. 왼쪽에서 **카테고리를 선택**합니다.
2. **「+ 링크 추가」** 버튼을 클릭합니다.
3. **제목**, **URL**, **설명(선택)**을 입력한 뒤 **「링크 저장」**을 누릅니다.

### 5.2 링크 수정

1. 링크 항목에 마우스를 올리면 **✏️ 버튼**이 보입니다.
2. ✏️를 클릭한 뒤, 제목·URL·설명을 수정하고 **「저장」**을 누릅니다.

### 5.3 링크 삭제

1. 링크 항목에 마우스를 올리면 **🗑️ 버튼**이 보입니다.
2. 🗑️를 클릭한 뒤, 확인하면 해당 링크와 **그 링크의 모든 메모**가 삭제됩니다.

### 5.4 링크 순서 변경

1. 링크에 마우스를 올리면 **⬆️** / **⬇️** 버튼이 보입니다.
2. **⬆️**로 한 칸 위로, **⬇️**로 한 칸 아래로 순서를 바꿀 수 있습니다.  
   (검색 결과 화면에서는 순서 변경 버튼이 나오지 않습니다.)

### 5.5 링크 선택 및 열기

- **링크 제목·URL 영역**을 클릭하면 해당 링크가 선택되고,  
  오른쪽 패널에 **그 링크의 메모 목록**이 표시됩니다.
- **URL**을 클릭하면 해당 사이트가 **새 탭**에서 열립니다.  
  (카테고리/링크 선택은 그대로 유지됩니다.)

---

## 6. 메모 (오른쪽 패널)

### 6.1 메모 추가

1. 가운데에서 **링크를 선택**합니다.
2. **메모 입력창**에 내용을 입력한 뒤 **「메모 추가」**를 누릅니다.

### 6.2 메모 수정

1. 메모 항목에 마우스를 올리면 **✏️ 버튼**이 보입니다.
2. ✏️를 클릭한 뒤, 내용을 수정하고 **「저장」**을 누릅니다.

### 6.3 메모 삭제

1. 메모 항목에 마우스를 올리면 **🗑️ 버튼**이 보입니다.
2. 🗑️를 클릭한 뒤, 확인하면 해당 메모만 삭제됩니다.

---

## 7. 처음 화면 / 명언

- **카테고리를 선택하지 않은 상태**에서는 가운데 패널에  
  **「나만의 즐겨찾기를 체계적으로 관리하세요」** 느낌의 **명언**이 표시됩니다.
- **로고를 클릭**해 처음 화면으로 돌아가면, **다른 명언**이 랜덤으로 보일 수 있습니다.
- 그 아래 **「왼쪽에서 카테고리를 선택하세요」** 문구로 다음 동작을 안내합니다.
