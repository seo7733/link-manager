위 버튼을 누른 뒤 인쇄 대화상자에서 「대상: PDF로 저장」 또는 「Microsoft Print to PDF」를 선택하면 PDF로 저장할 수 있습니다.

즐순이 - 즐겨찾기 매니저 사용자 매뉴얼

나만의 즐겨찾기를 카테고리별로 정리하고, 링크마다 메모를 남길 수 있는 웹 앱입니다.

1. 시작하기

1.1 로그인

  1. 사이트에 접속합니다.
  2. 「Google로 로그인」 버튼을 클릭합니다.
  3. Google 계정으로 로그인하면 메인 화면(대시보드)으로 이동합니다.

1.2 로그아웃

화면 오른쪽 상단의 「로그아웃」 버튼을 클릭하면 로그아웃됩니다.

2. 화면 구성

메인 화면은 헤더와 세 개의 패널로 이루어져 있습니다.

영역설명
헤더로고, 구글 검색·앱 검색, 이메일·로그아웃
왼쪽 패널카테고리 목록 (폴더 구조)
가운데 패널링크 목록 (선택한 카테고리의 링크)
오른쪽 패널메모 목록 (선택한 링크의 메모)

3. 헤더

3.1 로고

즐순이 로고를 클릭하면 처음 화면으로 돌아갑니다. (카테고리·링크 선택이 해제되고, 가운데에 명언이 다시 표시됩니다.)

3.2 구글 검색

입력창에 검색어를 입력한 뒤 🔍 버튼을 누르거나 Enter를 치면, Google 검색 결과가 새 탭에서 열립니다.

3.3 카테고리·링크·메모 검색

입력창에 단어를 입력한 뒤 🔍 또는 Enter를 치면, 카테고리 이름, 링크(제목·URL·설명), 메모 내용 중 해당 단어가 포함된 링크만 가운데 패널에 표시됩니다. 「검색 해제」 버튼으로 검색을 취소할 수 있습니다.

4. 카테고리 (왼쪽 패널)

추가: 「새 카테고리 이름」 입력 후 「추가」 클릭 또는 Enter.
하위 카테고리: 카테고리 옆 ➕ 클릭 후 이름 입력·추가.
수정: 카테고리에 마우스 올린 뒤 ✏️ 클릭 → 이름 수정 후 「저장」.
삭제: 🗑️ 클릭 후 확인 (하위 카테고리·링크·메모 모두 삭제됨).
순서 변경: ⬆️ / ⬇️ 버튼으로 같은 레벨에서 이동.
선택·펼치기/접기: 카테고리 이름 클릭 시 링크 목록 표시, ▶/▼로 하위 목록 접기·펼치기.

5. 링크 (가운데 패널)

추가: 카테고리 선택 → 「+ 링크 추가」 → 제목·URL·설명 입력 후 「링크 저장」.
수정/삭제: 링크에 마우스 올리면 ✏️·🗑️ 표시.
순서 변경: ⬆️ / ⬇️ 사용 (검색 결과 화면에서는 비표시).
선택·열기: 링크 제목·URL 영역 클릭 시 메모 패널 표시, URL 클릭 시 새 탭에서 사이트 열기.

6. 메모 (오른쪽 패널)

추가: 링크 선택 후 메모 입력창에 입력 → 「메모 추가」.
수정/삭제: 메모에 마우스 올리면 ✏️·🗑️ 표시.

7. 처음 화면 / 명언

카테고리를 선택하지 않으면 가운데 패널에 명언이 표시됩니다. 로고를 클릭해 처음 화면으로 돌아가면 다른 명언이 랜덤으로 보일 수 있습니다.
